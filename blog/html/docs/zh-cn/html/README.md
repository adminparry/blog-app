# html

> 基本结构

``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- 字符集 -->
    <meta charset="UTF-8">
    <!-- https://www.runoob.com/css/css-rwd-viewport.html -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- 标题 -->
    <title>Document</title>
    <!-- 外联css样式 -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    
</body>
<!-- 外联JavaScript -->
<script src="javascript.js"></script>
</html>
```
