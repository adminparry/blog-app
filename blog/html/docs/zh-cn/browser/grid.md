# center an element

> example

``` html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .container {
            height: 200px;
            display: flex;
            align-items: center;
            justify-content: center;
            border: 1px solid;
        }

        .item {
            border: 1px solid;
            width: 10em;
        }
    </style>
</head>

<body>
    <div class="container">
        <div class="item">I am centered!</div>
    </div>

</body>

</html>

```
